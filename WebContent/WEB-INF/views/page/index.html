  • 公司介绍

    展为技术

    展为智能是一家专为滑坡、崩塌、泥石流等地质灾害提供专群结合监测预警系统的整体解决方案供应商。方案架构包含现场智能监测设备、监测预警云服务平台、独有的数据算法解析，以及用户终端APP的远程操控单元。产品研发设计理念始终遵循“以人为本，科技防灾”的原则，依托国家级与省级地质灾害数据库互联互通和动态更新基础，为专群结合地质灾害监测预警工作提供稳定、可靠、符合国家标准的全流程数据服务支撑。
  • 设备介绍

    展为Z3（8合1）测量型GNSS接收机

    展为Z3型GNSS接收机是一款专为地质灾害监测研发的普适型仪器，本着“提高可靠性、提高集成度、降低功耗、降低成本”的原则，系统集成度高、功能优化，借助物联网传输技术，操作简单且维护方便。 对于我国已查明的28.8万余处地质灾害隐患点的防治工作，更需要普适型设备收集大量基础数据，进行成灾规律研究，识别早期隐患并实现精准预警。展为Z3普适型GNSS接收机是一款地灾场景专用的高精度表面位移监测设备。
  • 应用场景

    应用场景

    展为的数据联动系统以高精度GNSS接收机作为基站，进行所有监测信息的汇总和时间同步，表现在数据上就是后台所呈现的所有结果均来自于同一地点、同一时间，各设备间获得的数据进行交叉验证，可以准确地作为地灾研判的依据，减少由多数据融合还有一个优点就是能够全面的分析出当前监测区域的状况，比如通过天气分析雨量，通过地下水位评估土体受力情况，并进行一个长时间序列上的持续监测，为预警、边坡治理工作提供依据。
